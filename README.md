# AI-Assisted Market-Driven Career Optimization System

This project is a web-based system that analyzes resumes and compares user skills with real-time job market demand to provide career optimization insights.

The system extracts skills from resumes using AI-assisted parsing and compares them with job market data collected from external APIs.

## Key Features

- Resume upload and automated skill extraction
- AI-assisted resume parsing
- Job market data collection through external APIs
- Skill alignment analysis with market demand
- Career skill gap identification
- Role-based skill exploration

## Technologies

- Python
- FastAPI
- HTML
- CSS
- External APIs
- AI Models

## Project Status

This project is currently under development as part of a BS Computer Science final year project.
